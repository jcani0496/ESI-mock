# Artesanias GT - Mockup

## Descripcion
Mockup funcional sin backend para presentar la propuesta digital de Artesanias GT. El proyecto simula una tienda en linea de artesanias guatemaltecas con catalogo, detalle de producto, carrito, checkout y secciones informativas.

## Alcance del mockup
- Navegacion completa entre paginas.
- Catalogo con filtros y fichas de producto.
- Detalle de producto con galeria, historia y proceso.
- Carrito y checkout con flujo simulado.
- Secciones informativas de confianza, politicas y contacto.
- Datos cargados desde archivos locales en `src/data/productos.json`.

## Tecnologias
- React
- Vite
- CSS

## Requisitos
- Node.js LTS
- npm

## Instalacion y ejecucion
1. Instalar dependencias:
   `npm install`
2. Ejecutar en desarrollo:
   `npm run dev`
3. Generar version de produccion:
   `npm run build`
4. Previsualizar el build:
   `npm run preview`

## Nota
Este proyecto se mantiene como mockup funcional y no incluye backend.
